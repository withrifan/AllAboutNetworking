## Add Cisco Router to project PNETLab

- Go to Main menus
- Add new lab
- Input lab name & Click Add
- Right click in panel > Add Node
- Choose Cisco IOL
- Delete all script in Qemu Options & Save
- Right click on the Cisco Router node then Start
